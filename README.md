# 🎮 Python Mini Games Collection

A collection of beginner-friendly Python projects built to strengthen core programming concepts like loops, conditionals, file handling, and random number generation.

This repository demonstrates hands-on practice with Python fundamentals and logical thinking.

---

## 📌 Projects Included

### 1️⃣ 🔢 Number Guessing Game

A simple command-line game where the user tries to guess a randomly generated number within a given range.

**Concepts Used:**
- While loops
- Conditional statements
- Random module
- User input handling
- Basic game logic

---

### 2️⃣ 🧮 Simple Calculator with History

A command-line calculator that performs basic arithmetic operations and stores calculation history in a text file.

**Features:**
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- History saving
- Clear history option
- Error handling (division by zero, invalid input)

**Concepts Used:**
- Functions
- File handling (read, write, append)
- Conditional logic
- Input validation
- Program structure

---

### 3️⃣ 📰 Breaking News Generator

A fun Python program that generates random "breaking news" headlines using predefined lists of subjects, actions, and places.

**Concepts Used:**
- Lists
- Random selection
- Infinite loop with exit condition
- String formatting (f-strings)

---

## 🛠 Technologies Used

- Python 3
- Built-in Modules:
  - random
  - file handling

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/vaibhavchandrasati6-lab/Python-Games-Collection.git
```

2. Navigate into the folder:

```
cd Python-Games-Collection
```

3. Run any file:

```
python number_guessing_game.py
python simple_calculator.py
python breaking_news_generator.py
```
